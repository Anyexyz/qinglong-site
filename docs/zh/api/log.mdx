# 日志 API 文档

## 基础路径
`/logs`

## 接口列表

### 获取日志列表
```http
GET /
```

获取所有可访问的日志文件列表（排除黑名单文件）。

### 获取日志详情
```http
GET /detail
```

**查询参数**
```typescript
{
  path?: string,  // 日志路径
  file?: string   // 文件名
}
```

**响应**
- 成功返回日志内容
- 访问受限返回 403 错误

### 获取指定日志文件
```http
GET /:file
```

**查询参数**
```typescript
{
  path?: string  // 日志路径
}
```

**响应**
- 成功返回日志内容
- 访问受限返回
