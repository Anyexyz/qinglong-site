# 前置步骤

## 创建应用

在 系统设置 -> 应用设置中添加应用，选择模块权限，生成 `client_id` 和 `client_secret`

## 获取 Token

> POST /open/auth/token

### 查询参数

| 字段名称      | 字段类型 | 默认值   |
|-------------|--------|-------|
| client_id   | string | - |
| client_secret | string | - |

### 请求头参数

```json
{}
```

### 请求体参数

```json
{}
```

### 响应体参数

```json
{
  "code": 200,
  "data": {
    "token": "xxxxxx",
    "token_type": "Bearer",
    "expiration": 1237889999
  }
}
```
