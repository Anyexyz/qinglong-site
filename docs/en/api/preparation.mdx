# 前置步骤

## API 前缀
`http://localhost:5700/open`

## 创建应用

在 系统设置 -> 应用设置中添加应用，选择模块权限，生成 `client_id` 和 `client_secret`

## 获取 token

- **Endpoint:** `GET /auth/token`
- **Parameters:**
  - `client_id` (String, Required): Client ID
  - `client_secret` (String, Required): Client Secret
- **Responses:**
  - `200`: 返回用户密钥信息
    - `token` (String): 授权密钥
    - `token_type` (String): 标记类型
    - `expiration` (Integer): 有效期 默认30天
